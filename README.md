## 📂 Projects

### 🔹 IPL Match Data Analysis
Exploratory analysis of IPL match data to identify patterns in performance and match outcomes.

**Key Work:**
- Team win percentages  
- Toss impact on match results  
- Match outcome trends  

**Tools:**
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook  

---

### 🔹 Student Management System with Analytics & ML
A CLI-based application to manage student records with added analytics and basic machine learning.

**Features:**
- Add, view, search, update, and delete records  
- JSON-based data storage  
- Input validation and error handling  

**Analytics:**
- Total students, average, max, and min marks  
- Grade classification (A/B/C/Fail)  
- Grade distribution visualization  

**ML Integration:**
- Decision Tree model for grade prediction (based on marks)
